#Lab8_SwitchLEDinterface  

## Purpose
Learn to build a physical circuit.  

## What It Does
Use external switch to flash the external LED at 5Hz or leave it off. 

    if switch is pressed
        flash LED at 5Hz
    else
    	turn LED off
